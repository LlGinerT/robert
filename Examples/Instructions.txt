Full ROBERT workflows using the Robert_examples.csv database and the external test set Robert_example_test.csv. To run the workflows:

1) Go to the folder with the databases in a terminal with Python
2) Run the following command line from Command_line.txt:
python -m robert --ignore "['Name']" --y Target_values --csv_name Robert_example.csv --csv_test Robert_example_test.csv --names Name


The results of the workflow are stored in the /Examples/CSV_workflow/Results folder. A Jupyter Notebook to run the workflow is also included.

A full description of the workflow is included in the Examples section of the ReadTheDocs webpage (https://robert.readthedocs.io).
